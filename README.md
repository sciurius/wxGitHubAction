# wxGitHubAction

This repo is for GitHub Actions that build wxPerl.

It is intended mostly for Microsoft Windows systems, that need setting
up Strawberry Perl and wxWidgets first.

## Current state

* Strawberry Perl is installed
* wxWidgets are fetched (source) and built
* Alien::wxWidgets is added
* wxPerl is fetched and build

## Tested successfully with

* Windows Server 2025
* Strawberry Perl 5.42
* wxWidgets 3.2.9
* Alien::wxWidgets 0.73
* wxPerl 3.009/
